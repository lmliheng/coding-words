模糊搜索原理说明

1. 数据准备：首先需要一个数据数组，包含要搜索的所有项目。

// 示例数据数组
const data = [
  { id: 1, name: '苹果' },
  { id: 2, name: '香蕉' },
  { id: 3, name: '橙子' },
  // ... 更多数据
];

2. 监听用户输入：通过 input 事件监听用户在搜索框的输入。

// 监听输入框的 input 事件（每次输入都会触发）
searchInput.addEventListener('input', function(event) {
  // 获取用户输入的值，并转换为小写（实现不区分大小写搜索）
  const query = event.target.value.trim().toLowerCase();
  
  // 执行搜索并显示结果
  performSearch(query);
});

3. 执行模糊搜索：使用数组的 filter 方法和字符串的 includes 方法进行匹配。

function performSearch(query) {
  // 如果搜索词为空，显示所有数据
  if (query === '') {
    displaySearchResults(data);
    return;
  }
  
  // filter 方法遍历数组，对每个元素执行测试函数
  // includes 方法检查字符串是否包含指定的子字符串
  const filteredData = data.filter(item => {
    return item.name.toLowerCase().includes(query);
  });
  
  // 显示过滤后的结果
  displaySearchResults(filteredData);
}

4. 显示结果：动态创建列表元素显示搜索结果。

function displaySearchResults(results) {
  // 清空之前的搜索结果
  searchResults.innerHTML = '';
  
  // 如果没有匹配结果，显示提示信息
  if (results.length === 0) {
    const noResults = document.createElement('li');
    noResults.className = 'no-results';
    noResults.textContent = '没有找到匹配的结果';
    searchResults.appendChild(noResults);
  } else {
    // 遍历匹配结果，创建列表项
    results.forEach(item => {
      const li = document.createElement('li');
      li.className = 'result-item';
      li.textContent = item.name;
      searchResults.appendChild(li);
    });
  }
}

关键点：

• toLowerCase()：将字符串转换为小写，实现不区分大小写的搜索

• includes()：检查字符串是否包含指定子字符串，返回布尔值

• filter()：遍历数组，对每个元素执行测试函数，返回通过测试的元素组成的新数组

• trim()：去除字符串两端的空格，避免搜索" apple "时匹配不到"apple"
